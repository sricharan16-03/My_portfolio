## ✨ Features & Visual Highlights

*   **🌌 WebGL Particle Universe:** An immersive, responsive 3D WebGL particle system built using **Three.js** that tracks mouse movements to create a deep cosmic backdrop.
*   **🖱️ Dynamic Custom Pointer Follower:** A buttery-smooth, hardware-accelerated cursor follower with micro-animations, context-aware hover effects, and animated status text.
*   **⚡ Futuristic Preloader Gatekeeper:** An interactive, high-tech startup loader showing real-time file-compiling simulations and progressive logging metrics.
*   **🎭 Complex Motion Systems:** Scroll animations, slide panels, parallax card matrices, and responsive element transitions powered by **Framer Motion** and **GSAP**.
*   **🧊 Smooth Lenis Scroll:** Fully integrated buttery-smooth viewport physics utilizing **Lenis** to create premium reading inertia.
*   **💎 Glassmorphism UI:** Seamless translucent dashboard panels, HUD rings, scanline overlays, laser sweep scanners, and curated neon cyan-purple gradients.
*   **📱 Fully Responsive Design:** Responsive grid and flexbox layout architecture designed to scale seamlessly from ultra-wide displays to pocket smartphones.
*   **🔍 Optimized SEO:** Semantic HTML5 structure, descriptive meta declarations, descriptive IDs, and optimized font loading for perfect search engine performance.

## 💻 Local Development Setup

Follow these simple steps to download, install, and run this portfolio locally on your machine:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) (v18.0.0 or higher recommended) and `npm` installed.

### 2. Clone the Repository
```bash
git clone https://github.com/sricharan16-03/My_portfolio.git
cd My_portfolio
```

### 3. Install Dependencies
Install all required modules, including WebGL (Three.js), animations (GSAP, Framer Motion), and styling engines (Tailwind v4):
```bash
npm install
```

### 4. Launch Development Server
Launch the compiler and boot up the hot-reloading development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your favorite web browser to see the portfolio active!

### 5. Build for Production
Create an optimized, statically generated production bundle ready for Vercel/Netlify deployment:
```bash
npm run build
```
